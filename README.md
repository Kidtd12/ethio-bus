Ethio Bus 
 

A **React web application** that lets users **browse and explore bus companies in Ethiopia**. Users can click on each bus card to see full details, including images and descriptions. The app is clean, responsive, and easy to use.

[Live Demo](https://celebrated-klepon-c153a6.netlify.app)

---

## Features

- Browse all major bus companies in Ethiopia: Selam Bus, Habessa, Zemen, Abay, and more  
- Search and filter bus companies  
- View detailed company pages with images and descriptions  
- Responsive and mobile-friendly design  
- Built with Vite for fast performance  

---

## Tech Stack

- **Frontend:** React.js  
- **Styling:** Tailwind CSS  
- **Build Tool:** Vite  
- **Hosting:** Netlify
